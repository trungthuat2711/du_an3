Kéo dự án về máy tính: git pull

Nguyễn văn A: add code lần đầu và đưa thẳng lên git

-git init
-git add .
-git commit -m "first commit"
-git branch -M main
-git remote add origin https://github.com/trungthuat2711/TeamABTeamAB.git
-git push -u origin main

Nguyễn văn B: tạo thư mục dự án

-git init
-git remote add origin https://github.com/trungthuat2711/TeamABTeamAB.git
-git branch -M main
-git pull origin main
